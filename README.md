关于 GoodERP
----

面向年营业额在2千万到2亿元的增长型企业

需要按需定制的信息化系统支撑业务稳步增长

GoodERP是一款开源ERP软件

具备财务、人事、项目、采购、销售、库存等核心功能

不同于现有市场的套装软件或云软件

GoodERP不限用户数，模块化二次开发，源码和数据自主掌控，是开源专业化IT服务的载体。
